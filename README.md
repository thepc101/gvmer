# gvmer

The gaming operating system.

Premium, editorial, fashion-inspired desktop application for PC gaming.

## Tech Stack

- Next.js 14
- TypeScript
- TailwindCSS
- Framer Motion
- TanStack Query
- Electron
- LiveKit (voice)

## Development

```bash
npm install
npm run dev
```

## Desktop

```bash
npm run electron:dev
```

## Build

```bash
npm run electron:build
```
